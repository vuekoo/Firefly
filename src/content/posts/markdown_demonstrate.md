## 一.准备工作

1.下载插件
- Markdown All in One（编写文件）
- Markdown Preview Enhanced（将文件预览）
- Paste Image（插入图片）

2.创建一个.md文档，打开同步预览功能

## 二.基本语法

### 1.标题
- `#` 一级标题
- `##` 二级标题
- `###` 三级标题
- `####` 四级标题
- `#####` 五级标题
- `######` 六级标题

标题一共有六种等级。从一到六，标题数字越大，显示的字符就越小。

### 2.引用
```
> ">"用于编辑文档十分好用
>> ">>"嵌套编辑的文档
```

### 3.列表
- 减号
+ 加号
* 乘号

有序列表：前面加 1. 2. 3.

嵌套列表：前面空四个格子就可以实现列表的嵌套
